# Python_MQTT_Client
Python MQTT client implementation: lightweight client supporting connect/disconnect, publish/subscribe, QoS 0-2, and Ping Request.

# Features:
* CONNECT / DISCONNECT
* PUBLISH / SUBSCRIBE (QoS 0/1/2)
* PINGREQ
# Limitations:
* No authentication, no TLS, no will messages, no session expiry, no topic aliases, etc.
* No support for MQTT 5 features like user properties, reason codes, etc.
